Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It's the foundation of all websites and describes the structure of a web page. HTML consists of a series of elements that tell the browser how to display content.

HTML was created by Tim Berners-Lee in 1991, and it has gone through many versions since then. The current standard is HTML5, which introduced many new features and improvements over previous versions.

HTML is not a programming language; it's a markup language that defines the structure of your content. It uses tags to define elements within a document.

HTML Document Structure

Every HTML document follows a basic structure. Here's what a minimal HTML document looks like:

HTML
Result
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First HTML Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
</body>
</html>

Hello, World!

This is my first HTML page.

Let's break down this structure:


<!DOCTYPE html>

The DOCTYPE declaration is not an HTML tag; it's an instruction to the web browser about what version of HTML the page is written in. The declaration <!DOCTYPE html> represents HTML5.

<html> Element

The <html> element is the root element of an HTML page. All other elements are contained within this element. The lang attribute specifies the language of the document.

<head> Element

The <head> element contains meta-information about the document, such as its title, character encoding, styles, scripts, and other metadata that isn't displayed directly on the web page.

Important elements inside the head include:

  • <meta>: Provides metadata about the HTML document
  • <title>: Specifies the title of the document (shown in browser tab)
  • <link>: Links to external resources, like CSS files
  • <script>: Embeds or references JavaScript code

<body> Element

The <body> element contains all the content that is displayed on the web page, such as text, images, links, tables, lists, etc.

HTML Elements, Tags, and Attributes

Elements vs. Tags

An HTML element is defined by a start tag, some content, and an end tag. For example: 

<p>This is a paragraph.</p>

In this example, <p> is the start tag, "This is a paragraph." is the content, and </p> is the end tag. Together, they form the HTML element.

Attributes

HTML attributes provide additional information about an element. They are always specified in the start tag and usually come in name/value pairs like: name="value". 

<a href="https://www.example.com">Visit Example.com</a>

In this example, href is an attribute of the <a> element, and its value is "https://www.example.com".

Empty Elements

Some HTML elements have no content and don't need an end tag. These are called empty elements or self-closing elements. 

<img src="image.jpg" alt="Description of image">

In HTML5, the trailing slash is optional for empty elements. Both <img src="image.jpg" alt="Description"> and <img src="image.jpg" alt="Description" /> are acceptable.
